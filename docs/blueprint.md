# **App Name**: Mura

## Core Features:

- Hybrid Source Search: A unified search that queries both YouTube Music and Spotify APIs simultaneously for a comprehensive library experience.
- Nested Music Folders: Deep hierarchical organization for playlists and tracks, allowing for advanced library management without flat-list limitations.
- Clean-Stream Ad-Blocker: Backend logic designed to fetch direct audio streams to provide a seamless listening experience without intrusive visual or audio ads.
- Performance-First Modular UI: A lightweight three-pane interface inspired by Cinny, optimized for slow devices to ensure lag-free navigation between folders and queues.
- Community Library Sync: Sync folder structures and library integrations across devices using a lightweight database backend for reliability.
- Optional AI Curation Tool: A fully optional tool that uses reasoning to suggest tracks for your folders based on metadata patterns if the user enables it.
- Tactile Drag-and-Drop: Intuitive curation gestures to move tracks between platforms and nested folders with ease.

## Style Guidelines:

- Primary color: Deep Amethyst (#6D28D9), providing a friendly and vibrant community feel.
- Background: Midnight Obsidian (#0A0A0B), a clean, eye-friendly black that is softer than pure black.
- Accent: Shadow Cobalt (#1E3A8A), a dark blue used for subtle pane separation and depth.
- Primary font: 'Lexend' for its highly legible, friendly, and non-corporate aesthetic.
- A modular multi-pane design that blends Spotify's rounded card aesthetic with YouTube Music's minimal content focus.
- Lightweight CSS-based transitions that mimic the fluid, spring-physics feel of Cinny without taxing older hardware.
- Soft-edged, rounded line icons that feel approachable and community-oriented.
